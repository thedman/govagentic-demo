# Govagentic Demo

Public portfolio demonstration of AI governance concepts applied to third-party risk assessment.

## Portfolio Demo Notice

This repository uses synthetic vendor, control, assessment, and audit data. It is not connected to production systems or real client environments.

No secrets or credentials are intended to be stored in this repository. Any live API credentials used by the demonstration must be stored server-side in the hosting platform's secret manager and rotated outside the public codebase.

The demo preserves enough detail to show architecture and reasoning quality while avoiding publication of a complete proprietary governance methodology.
